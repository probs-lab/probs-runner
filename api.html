Top-level functions¶

probs_runner.probs_convert_and_query_data(datasources, queries, print_facts=False, working_dir=None, script_source_dir=None) → Dict¶

Run RDFox to load facts and rules then return answer from queries.

Parameters

  • facts – Facts in ttl format (string)

  • rules – Facts in datalog format (string)

  • queries – Dict of {query_name: query_text}, or list of [query_text].

  • print_facts – whether to dump all RDFox facts for debugging

  • working_dir – Path to setup rdfox in, defaults to a temporary directory

  • script_source_dir – Path to copy scripts from

Returns

Dict of {query_name: result}
